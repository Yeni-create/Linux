# Linux Fundamentals for PostgreSQL DBAs

This folder contains Linux commands, notes, and examples that are essential
for managing PostgreSQL databases in production environments.

The focus is on practical system administration skills used by PostgreSQL DBAs,
including service management, logging, networking, storage, and performance
monitoring.

## Topics Covered
- systemd and service management
- OS and PostgreSQL-related log analysis
- Networking basics for database connectivity
- Disk, memory, and swap management
- Performance monitoring and troubleshooting

## Platforms
- Rocky Linux 8/9
- RHEL-based Linux distributions
